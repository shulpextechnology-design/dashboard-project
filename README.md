# Bharat Tools Hub Dashboard

This project is a dashboard application with a client extensions management system.

## Project Structure

- **client/**: React frontend application.
- **server/**: Node.js/Express backend server with SQLite database.
- **extension_analysis/**: Contains analysis or source code for the browser extension.

## Getting Started

1.  **Server**:
    ```bash
    cd server
    npm install
    node index.js
    ```

2.  **Client**:
    ```bash
    cd client
    npm install
    npm run dev
    ```

## Features

- User Authentication (Single Session Enforcement)
- Admin Panel for User Management
- Extension Integration with Helium 10 Session Sync
